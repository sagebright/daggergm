## Key Workflow Principles

### 1. Progressive Disclosure

- Start simple: Generate outline (1 credit)
- Add complexity: Confirm scenes one by one
- Full control: Expand with details when ready

### 2. Clear Progress Indicators

- Visual confirmation badges (âœ…/â¬œ)
- Progress bar (2/3 scenes confirmed)
- Regeneration budget (3/10 used)

### 3. Explicit User Control

- No auto-progression
- User confirms each scene manually
- User decides when to mark as ready
- User chooses when/if to expand

### 4. Non-Destructive Editing

- Regenerate doesn't delete - it replaces
- Can unconfirm to re-edit
- Export is additive, not destructive

### 5. Budget Awareness

- Credits shown at all times
- Regeneration limits visible
- Cost shown before generation
